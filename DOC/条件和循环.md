# if语句

```
if expression:
    expr_true_suite
```

# else语句

```
if expression:
    expr_true_suite
else:
    expr_false_suite
```

# elif语句

`elif`语句是Python的`else-if`语句，它检查多个表达式是否为真，并在为真是执行特定代码块中的代码。if语句最多只能有一个else语句，但是
可以有任意数量的elif语句。

```
if expression1:
    expr1_true_suite
elif expression2:
    expr2_true_suite
    .
    .
elif expressionN:
    exprN_true_suite
else:
    none_of_the_above_suite
```
